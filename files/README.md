# Café com Lenita — Site institucional

Projeto estático em HTML, CSS e JavaScript para apresentar a Comunidade Café com Lenita.

## Arquivos

- `index.html` — estrutura da página
- `style.css` — estilos visuais e responsividade
- `script.js` — menu mobile e animações suaves de entrada
- `foto1.webp` — imagem utilizada na seção principal

## Ajustes recomendados

Antes de publicar, substitua:

- `https://wa.me/5500000000000` pelo número real do WhatsApp
- `https://go.hotmart.com/SEU-LINK` pelo link real da Hotmart
- depoimentos ilustrativos por depoimentos reais
- placeholders de mídia por fotos e vídeos finais

## Como visualizar

Basta abrir o arquivo `index.html` no navegador.
